# iris-flower-classification
Machine Learning project to classify Iris flowers into Setosa, Versicolor, and Virginica species using Python and Scikit-learn
# Iris Flower Classification

This project uses Machine Learning algorithms to classify Iris flowers based on their sepal and petal measurements.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Algorithm
- Logistic Regression

## Dataset
Iris Dataset from Scikit-learn

## Output
Predicts flower species:
- Setosa
- Versicolor
- Virginica
